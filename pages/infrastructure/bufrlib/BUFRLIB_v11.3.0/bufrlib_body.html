The Binary Universal Form for the Representation of meteorological data (BUFR) is a binary data format maintained by the World Meteorological Organization (WMO). NCEP maintains a BUFR code library (BUFRLIB) for use in all NCEP Production Suite applications that read/write data in BUFR format. BUFRLIB is part of the NCEP Common Software (NCEPLIBS) project; the latest version of the BUFRLIB software is maintained on GitHub at https://github.com/NOAA-EMC/NCEPLIBS-bufr. The latest documentation can be found at https://noaa-emc.github.io/NCEPLIBS-bufr/.
